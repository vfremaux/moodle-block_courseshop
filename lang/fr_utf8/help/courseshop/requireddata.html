Données clientes associée au produit

Certains produits, pour être réalisés en production nécessitent des données complémentaires que le client doit fournir. Ce paramètre permet de définir leurs nom et leur type et leur description. Les données à fournir seront demandées sur le formulaire de confirmation de commande.

<paramname 1>:<paramtype 1>:<paramdesc 1>;<paramname 2>:<paramtype 2>:<paramdesc 2>...

Les paramètres, noms et descriptions ne doivent contenir ni ':', ni ';';

Noms de paramètre

Les noms de paramètres doivent ne contenir ni espace, ni caractères spéciaux. Il est conseillé d'éviter les caractères accentués.

Types des paramètre récoltés

Le type des paramètres récoltés peut être :

  • textfield : Une ligne de texte simple;
  • checkbox : une option booléenne;
  • textarea : Un paragraphe de texte;
